# 🚗 Ride Cancellation Prediction System

This project is a machine learning-based web application that predicts ride cancellation risk using real-world ride booking data.

## 📌 Overview

The system allows users to input ride details and get real-time cancellation probability.

## 🚀 Features

* Interactive Streamlit dashboard
* Real-time prediction
* Random Forest ML model
* Clean UI

## 🛠️ Tech Stack

Python, Pandas, NumPy, Scikit-learn, Streamlit

## ▶️ Run

```bash
streamlit run app.py
```

## 🎯 Objective

Predict ride cancellations to improve decision-making.
